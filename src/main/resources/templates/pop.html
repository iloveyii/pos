Waiting Popup Demo

Click the button below to see a non-closable waiting popup for 3 seconds

How it works:
// Show the modal
const modal = new bootstrap.Modal(document.getElementById('waitingModal'));
modal.show();

// Disable closing
document.getElementById('waitingModal').addEventListener('hide.bs.modal', function (e) {
    e.preventDefault();
});

// Hide after 3 seconds
setTimeout(function() {
    modal.hide();
}, 3000);
