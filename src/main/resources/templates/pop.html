How to use:

  1. Click the "Show Waiting Popup" button
  2. A modal will appear with a spinner and countdown timer
  3. The modal cannot be closed by clicking outside or pressing ESC
  4. After 3 seconds, the modal will automatically close
  5. You'll see a success message when the modal closes
Implementation Code:
// Show the modal
const modal = new bootstrap.Modal(document.getElementById('waitingModal'));
modal.show();

// Set the countdown timer
let seconds = 3;
const countdownElement = document.getElementById('countdown');
countdownElement.textContent = seconds;

const countdownInterval = setInterval(function() {
    seconds--;
    countdownElement.textContent = seconds;
    if (seconds <= 0) clearInterval(countdownInterval);
}, 1000);

// Hide after 3 seconds
setTimeout(function() {
    modal.hide();
    clearInterval(countdownInterval);
}, 3000);
